# Práticas de Lógica de Programação em C# 🚀

Este repositório contém uma coleção dos meus exercícios pessoais de lógica de programação, implementados utilizando a linguagem C#. O objetivo principal é fortalecer minhas habilidades de resolução de problemas 🧠 e aprofundar o entendimento dos fundamentos da programação em C# 💻.

## O que você encontrará aqui 👇

Aqui você encontrará uma variedade de exercícios que exploram conceitos como:

* Estruturas de controle (`if`, `else`, `switch`, `for`, `while`, `do-while`) 🚦
* Manipulação de dados (variáveis, tipos primitivos e de referência, arrays, listas (`List<T>`)) 🗃️
* Lógica booleana e operadores (aritméticos, de comparação, lógicos) 🤔
* Algoritmos básicos (ordenação simples, busca, etc.) 💡

## Como explorar 🔍

Sinta-se à vontade para navegar pelas pastas e arquivos. Cada exercício geralmente estará em um arquivo `.cs` separado, com comentários explicando a lógica implementada.
